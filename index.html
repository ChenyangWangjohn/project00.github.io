Part 1 — Portrait Series at Controlled Camera Offsets

Distances are expressed relative to a baseline vantage point that yields the desired framing. Negative values indicate moving closer than baseline; positive values indicate stepping back. When stepping back, I increased focal length to keep face size approximately constant.

Reflection — Part 1

When stepping closer to the person, his facial features become more pronounced, with the nose appearing larger relative to the ears. When stepping back, the facial features appear more compressed and proportional. I think this is because when moving closer, the proportional distance differences between the noses and ears gets exaggerated. Therefore, the nose and mouth occupy a larger visual space, drawing more attention.

Reflection — Part 2

Note: Edits typed here are not saved to disk; copy them into the HTML before submission if needed.

Similar as Part 1, moving the camera changes perspective; changing focal length only changes framing. When I get close, near parts of the scene are much closer than far parts, so they project disproportionately larger. When I step back, those distance differences shrink, so the scene looks more “neutral” or “compressed.”
